                                              XOX Tracker
XOX Tracker is a smart market analysis dashboard designed to help users track price fluctuations on major e-commerce platforms. It provides real-time visual insights, helping buyers decide the perfect time to purchase.

✨ Features
-Dynamic Price History Chart: Visualizes price movements over time to identify market volatility.

-Interactive Activity List: Allows users to toggle between different products to view specific trends.

-Instant Price Insights: Automatically calculates and displays price drops ("Save") or hikes ("Up") relative to previous checks.

-Multi-Platform Support: Ready for integration with Trendyol, Amazon, Hepsiburada, and Alibaba.

-Optimized Performance: Built with a focus on resource efficiency for lightweight environments like antiX Linux.

🛠️ Tech Stack
-Frontend: React.js, Tailwind CSS, Lucide Icons (conceptual).

-Backend: FastAPI (Python), Selenium WebDriver (Scraping Engine).

-Database: SQLite (SQLAlchemy).

  Prerequisites
-Node.js (v18+)

-Python 3.10+

-Google Chrome & ChromeDriver (for Selenium)
